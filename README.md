# Vintin
Vinting like a baws.

How to Vint:
1. Go to \Vintin\src\ConsoleApp\
2. run: "dotnet run --project ConsoleApp.csproj"
3. or run: "dotnet run --project ConsoleApp.csproj input.txt"

How to Test Vintin:
1. Go to \Vintin\
2. run: "dotnet test"
